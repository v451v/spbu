# СПБУ Foundation — Расчёт оснований самоподъёмных плавучих буровых установок

Система расчёта несущей способности и осадок оснований фундаментов СПБУ для морских нефтегазовых платформ.

## Возможности

- ✅ **Две методики расчёта**: Российская (СП 22/23/58) и Западная (SNAME/ISO 19905-1)
- ✅ **Веб-интерфейс** на Streamlit для удобного ввода данных
- ✅ **Интерактивные графики** с визуализацией кривых пенетрации
- ✅ **Импорт/экспорт** конфигураций в формате TOML

## Запуск

### Streamlit UI (веб-интерфейс)

```bash
uv run streamlit run app.py
```

Приложение откроется в браузере по адресу http://localhost:8501

### CLI (командная строка)

```bash
# Российская методика
uv run python main.py input.toml

# Западная методика  
uv run python main.py input_western.toml
```

## Быстрый старт

1. Запустите: `uv run streamlit run app.py`
2. Выберите методику в боковой панели
3. Введите данные: слои грунта, фундамент, нагрузки
4. Нажмите "Рассчитать"
5. Изучите результаты
